# 🔍 GetRepos 

## 📌 Overview
GetRepos is a dynamic, front-end web application built with Vanilla JavaScript. It interacts directly with the GitHub REST API to allow users to search for any GitHub profile and instantly retrieve a list of their public repositories. This project demonstrates practical experience with asynchronous programming, API integration, and dynamic UI rendering.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling for a clean, user-friendly search interface and result cards.
* **JavaScript (ES6+):** Core logic, utilizing modern syntax including arrow functions, object destructuring, and template literals.

## ✨ Key Features
* **Live API Integration:** Fetches real-time data using the `https://api.github.com/users/{username}/repos` endpoint.
* **Dynamic DOM Manipulation:** Programmatically creates HTML elements (cards, links, badges) based on the parsed JSON data returned from the API.
* **Error Handling:** Gracefully handles invalid usernames or empty states to ensure a smooth user experience.
* **Data Extraction:** Extracts and displays specific repository details, such as the repo name, direct URL, and star/fork counts.

## 🧠 Technical Learnings
Building GetRepos was a deep dive into modern JavaScript. It solidified my understanding of the fetch API, handling HTTP responses, parsing JSON objects, and efficiently updating the Document Object Model (DOM) without requiring page reloads.
